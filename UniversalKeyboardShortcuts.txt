Keyboard Shortcuts
  GENERAL
  	Navigate focus     
  		Super + hjkl 
  	Move Window
		Ctrl+Super+hjkl
  	Toggle Fullscreen / Maximize
  		Super + D
  	Toggle floating window
		Super + T
  	Close Window
		Super + Backspace / F4 (windows)
  LAYOUT
	Stack Windows
		Super + I / Arrowkeys (windows)
	Unstack Windows
		Super + O 
	Resize Window Horizontal
		Super + R / +- (windows)
	Resize Window Vertical
		Super + Shift + R / +- (windows)
  MULTIMONITOR
  	Navigate through Monitors
		Shift (debian) + Super + hjkl
  	Move Windows through Monitors
		Ctrl + Shift (debian) + Super + hjkl
  WORKSPACE
  	Navigate through Workspaces
		Super+PgUp/Down
 	Move Window through Workspaces
		Ctrl+Super+PgUp/Down
  LAUNCHER
	Launch new Terminal
		Ctrl + Alt + Plus

	disable all gnome shortcuts with super key and gnome tiling shortcuts
